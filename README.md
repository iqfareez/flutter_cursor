# Flutter System Cursors

## Getting Started

- Setup Flutter on your machine if you haven't already. Follow the instructions [here](https://flutter.dev/docs/get-started/install).
- Run `flutter run` in the terminal to start the app.

## Releases

Try the app on the web: https://flutter-system-cursors.netlify.app/

Compiled binaries for different platforms are available on the [Releases](https://github.com/iqfareez/flutter_cursor/releases) page.
